How Does a Countries GDP and total Volunteers affect their Clean up efficiency when looking at regions on Earth

Context

For the second research question, similar to the analysis in research question 1 it is also limited to countries included in the Tidy Tuesday plastics data set containing data from 2019. Due to this limitation our data will not be applicable to all countries and will be limited to that of the plastics data set. The Question sparked into our minds by wondering about different Regions and country sizes in the data-set. These regions and countries in the data set don’t all have the same population sizes, some are drastically larger than other, thinking about the volunteer sizes, if a country has more volunteers would their clean up efficiency be better? Prior to actually finding evidence, there was a theory, that perhaps those countries with higher volunteer sizes would have a better cleaning efficiency. Now alongside that there was another variable GDP, logically thinking wouldn’t those with higher GDP’s also have better cleaning efficiency due to more money and resources being available? This question was taken a step further and wanted to determine if there was an interaction between GDP, Volunteer size, and cleaning efficiency. This question was brought back to one variable, regions. How do regions interact with these variables, now not all regions of the world have the same population sizes as stated earlier, neither do they have the same amount of volunteers or money because of there is now a way that could draw us to an even more statistically significant conclusion. As expect regions such as Asia, Europe and the Americas to have the highest cleanup efficiency due to their higher populations and overall GDP’s.

get_region <- function(region) {
  url <- glue(
    "https://restcountries.com/v3.1/region/{region}?fields=name,population,region"
  )
  res <- GET(url)
  stop_for_status(res)
  txt <- content(res, as = "text", encoding = "UTF-8")
  df <- fromJSON(txt, flatten = TRUE)
  as_tibble(df) %>%
    transmute(
      country_api = coalesce(name.common, name.official),
      region = region,
      population = population
    )
}

empty_gdp <- tibble(
  country = character(),
  year = integer(),
  gdp_per_capita_nominal = numeric()
)

get_GDP <- function(country, key = my_key) {
  req <- request("https://api.api-ninjas.com/v1/gdp") |>
    req_url_query(country = country, year = 2019) |>
    req_headers(`X-Api-Key` = key)

  resp <- tryCatch(req_perform(req), error = function(e) NULL)
  if (is.null(resp)) {
    return(empty_gdp)
  }

  out <- tryCatch(
    resp_body_json(resp, simplifyVector = TRUE),
    error = function(e) NULL
  )
  if (is.null(out) || length(out) == 0) {
    return(empty_gdp)
  }

  as_tibble(out) %>%
    select(any_of(names(empty_gdp)))
}

cleanupbyregion <- plastic_with_region %>%
  filter(!is.na(region), !is.na(num_events), num_events > 0) %>%
  split(.$region) %>%
  map_dfr(function(df) {
    tibble(
      region = first(df$region),
      total_plastic = sum(df$grand_total, na.rm = TRUE),
      total_events = sum(df$num_events, na.rm = TRUE),
      avg_efficiency = total_plastic / total_events
    )
  }) %>%
  arrange(desc(avg_efficiency))
